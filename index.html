## Projects

### 1. N back / Dual N back (Python game)

This project was made after 'N back task' which helps improve working memory and cognitive skills.

**What I learned**

  - Gained an experience in Object Oriented Programming in Python, and gave me a chance to think about the importance of architectural codes, scalability.

  - Gained an experience in learning Game Development, such as time controls, and constant game loop.

**Game Download**

Download ZIP at [Repository](https://github.com/kjs29/dual_nback)

If you are a mac user, you can simply copy & paste the following code into your terminal.

The game will be downloaded in `~/Desktop/pygame_dual_nback`.

```

cd ~/Desktop && mkdir pygame_dual_nback && cd pygame_dual_nback && python3 -m venv env && source env/bin/activate && git clone https://github.com/kjs29/dual_nback.git && cd dual_nback && pip install -r requirements.txt && python main.py && deactivate

```
