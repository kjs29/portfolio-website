### 1. N back / Dual N back (Python game)

This project was made after 'N back task' which helps improve working memory and cognitive skills.

  - Gained an experience in Object Oriented Programming in Python, and gave me a chance to think about the importance of architectural codes, scalability.

  - Gained an experience in learning Game Development, such as time controls, and constant game loop.

**Game Download**

Download ZIP at [Repository](https://github.com/kjs29/dual_nback)

If you are a mac user, you can simply copy & paste the following code into your terminal.

The game will be downloaded in `~/Desktop/pygame_dual_nback`.

```

cd ~/Desktop && mkdir pygame_dual_nback && cd pygame_dual_nback && python3 -m venv env && source env/bin/activate && git clone https://github.com/kjs29/dual_nback.git && cd dual_nback && pip install -r requirements.txt && python main.py && deactivate

```

Projects

1. Dual N back | Python | Download & Code

Participated and won $50 in Pygames Hackathon by Microsoft

This project was made after 'N back task' which helps improve working memory and cognitive ability.

  • I had lots of fun building the game. While tutorials can provide a foundation, this project underscored the idea that true learning comes through application. It also boosted my confidence in programming.
  • Besides improving my problem-solving skills, I acquired experience in various aspects of game development, including implementing pause/resume functionalities, and stop watch-style time controls within the game loop.
  • I gained an experience in object oriented programming in Python, and why object oriented programming is suitable for complex codes.
  • As the codes get more complex, I came to deeply think about writing scalable, readable, and clean codes.

2. Portfolio website | HTML, CSS, JavaScript | Live | Code

It is a website for portfolio purpose. It is the current website you are looking at. The website which models after the MacOS environment is made with HTML, CSS, and vanilla JavaScript.

  • I strengthened my frontend skills, especially responsive design and Flexbox in CSS, and JavaScript Document Object Model(DOM) within Web API.
  • I learned and implemented good practices such as
    • Avoid global queries, as document.getElementById() traverses the entire DOM tree.
    • Manipulating classes for styles instead of adding inline styles. This helps with performance, readability, and maintainability.
